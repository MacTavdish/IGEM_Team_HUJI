Instructions for donwloading and be set up for README.txt tutorial :

1. Find the version by the name of zipped file in GitHub repository
or DTU-DENAMRK 2016 Codon Optimization tab
2. Download and extract in a folder of your preference
3. After extraction a folder named TaiCO will be generated
4. Enter the folder and carefully read README.txt file on how to properly
run TaiCO software
